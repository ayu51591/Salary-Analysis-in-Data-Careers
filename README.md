# Salary Analysis in Data Careers

## Objective
To analyse how job role, experience, and industry influence salary in data-related careers.

## Dataset
- Source: Job Salary Prediction Dataset
- Features: job role, experience, industry, salary, etc.

## Key Insights
- Role has a stronger impact on salary than industry
- Data Scientists earn significantly more than Data Analysts at similar experience levels
- Data Analyst salaries tend to plateau with experience
- Industry affects job availability more than salary differences

## Tools Used
- Excel (Pivot Tables)
- SQL (for structured queries)

## Conclusion
Students aiming for higher salary growth should focus on role specialisation (e.g., Data Science) rather than relying only on industry selection.

## Limitations
- No time-based trends
- Possible synthetic data inconsistencies
